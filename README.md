# PythonLesson

このレポジトリはPython初学者向けの教育資料になります。

全8回の構成で、各ノートブックを見ながら、GoogleColabで実行していくことで

Pythonの基本的な考え方、データ分析に必要最低限となるスキルを身に付けてもらいます。

※各回にノートブックのリンクを貼っています。お使いください。
